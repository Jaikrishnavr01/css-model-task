HTML Terms

HTML
HyperText Markup Language - a standard markup language for creating web pages.
HTML element
An individual component or part of an HTML document, consisting of a start tag, content, and an end tag.
HTML tag
Markup used to define the structure of an HTML document. It consists of an opening tag, content, and a closing tag.
Angle bracket
The characters < and >, used to enclose HTML tags.
HTML attribute
Additional information provided to HTML tags to modify their behavior or appearance.
HTML comment
Text in an HTML document that is not displayed but can be seen in the source code. It is used for adding notes or comments.
HTML object element
An HTML element used to embed objects, such as images, audio, or videos, into a web page.
HTML special character
Reserved characters in HTML that have a special meaning, such as < for < and > for >.
